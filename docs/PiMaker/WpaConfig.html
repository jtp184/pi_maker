    build() click to toggle source

    Adds together the preamble and country code to the networks to make the completed file contents string

    # File lib/pi_maker/wpa_config.rb, line 78
def build
  "#{PREAMBLE}country=#{country_code}\n\n" + networks.map.with_index do |n, x|
    format(NETWORK_FORMATTER, ssid: n[0], password: n[1], priority: x + 1)
  end.join("\n\n")
end
